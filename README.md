## 🎨 Design System & Specifications
*   **Layout:** Classic Messenger Layout with sticky bottom input panel.
*   **Typography:** Inter (Highly readable text for long conversational threads)
*   **Color Palette:**
    *   User Bubble: `#3B82F6` (Trust Blue)
    *   AI Bubble: `#E5E7EB` (Light Neutral Gray)
    *   Background: `#F3F4F6`

---

## 💡 UX Solutions Implemented
*   **Readability & Contrast:** Differentiated user and AI responses with distinct background shapes, color tones, and left/right alignments.
*   **Scroll Management:** Implemented auto-scroll behaviors to always keep the freshest incoming AI responses within the viewport.
*   **Actionable UI:** Included quick-tap suggestion pills to guide user intents without typing.
